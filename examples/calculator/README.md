# Calculator example

This is a simple calculator example

Like every example, you have a re-creation of the same project in web-based. Here's the comparison:
## Lines count
Lines (Web): 40 (HTML) + 88 (JavaScript) + 30 (CSS) => 158
Lines (Rust Native): 38 (XML) + 171 (Rust) + 26 (CSS) => 235
It's 148% bigger in the Rust version

## Chars count
Chars (Web): 1618 (HTML) + 2025 (JavaScript) + 351 (CSS) => 3994
Chars (Rust Native): 1547 (XML) + 5441 (Rust) + 311 (CSS) => 7299
It's 182% bigger in the Rust version

## RAM usage
RAM (Web): 500MB -> Chromium (Brave)
RAM (Rust Native): 89.9MB
The web version uses 556% more RAM

Rust version: rustc 1.95.0
